|---|---|---|---|
| 1 | Auth modal (email + social login) | Onboarding | Interactive entry card with blurred overlays |

| 3 | 3-step onboarding wizard | Onboarding | Post-registration details gather (name → use cases → preferences) |
| 4 | Protected routes | App-wide | Route guarding to redirect unauthorized requests to Login |
| 5 | Sidebar with conversation list | Home + Chat | Navigation panel containing active/past chat listings |
| 6 | Animated welcome screen | Home | Contains central glowing core orb and personalized greetings |
| 7 | Suggestion cards | Home | 2x2 clickable grid to start conversations instantly |
| 8 | Chat input area | Home + Chat | Centered textbox with auto-resizable 1-6 rows |
| 9 | Chat message thread | Chat | Left/right aligned message bubbles displaying user and AI turns |
| 10 | AI typing indicator | Chat | 3-dot staggered bounce animation while loading responses |
| 11 | Markdown rendering | Chat | Support for full rich-text Markdown with HTML structure |
| 12 | Syntax-highlighted code blocks | Chat | Multi-language code boxes with instant copy badges |
