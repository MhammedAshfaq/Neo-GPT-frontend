# 💼 Project Overview

**Neo GPT** is a premium, production-grade AI chat assistant web application. It is visually inspired by ChatGPT but built with a unique **Neo identity** — futuristic, dark, and purple-accented.

## Application Purpose

| Aspect | Description |
|---|---|
| **Who uses it** | Anyone who wants to chat with an LLM AI |
| **What it does** | Conversational AI interface with chat history, markdown, code highlighting |
| **How it works** | Frontend talks to a backend API that connects to an LLM |
| **Where it runs** | Browser (desktop + mobile responsive) |

---

## Key Differentiators from ChatGPT

| Feature | ChatGPT | Neo GPT |
|---|---|---|
| **Brand Identity** | OpenAI green/black | Neo purple/dark |
| **Auth Flow** | Modal login | Glassmorphism modal + 3-step onboarding wizard |
| **Welcome Screen** | Simple prompt | Animated orb + personalized greeting + suggestion cards |
| **Design System** | Minimal | Glassmorphism + glow effects + micro-animations |
