# 🔮 Visual Identity & Philosophy

Neo GPT is designed to be **futuristic, premium, and responsive**. It provides an interface that balances high-tech elements with dark readability.

---

## 1. Visual Identity Principles

### Dark-First Canvas
All views are rendered on a solid dark slate backdrop (`#0a0a0f`). This deep-space layout reduces visual fatigue, increases contrast for accent highlights, and feels modern.

### Electric Purple Accents
We use violet gradients and halos (`#7c3aed` to `#9070ff`) to represent intelligence, creativity, and AI actions. Highlights are applied to active buttons, cursor blinks, selection states, and loading flows.

### Glassmorphism Surfaces
Containers like sidebars, onboarding cards, and popup panels utilize translucent backdrops (`rgba` values) coupled with heavy background filters (`backdrop-filter: blur()`). This allows layered depth across screens.

### Glows & Halos
Interactive elements (such as log-in buttons and core welcome orbs) are framed with radial drop shadows. This simulates three-dimensional glow effects.

### Dynamic Responses
The interface reacts to user interactions. Hover triggers prompt translations, arrows slide, list items highlight, and panels expand.

---

## 2. Mood Reference

* **Theme:** Deep space + electric purple
* **Personality:** Intelligent, trustworthy, responsive, premium but approachable.
* **Goal:** A sleek interface that performs fluidly.
