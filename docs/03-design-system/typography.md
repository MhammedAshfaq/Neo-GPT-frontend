# 🔤 Typography System

Neo GPT employs clear and modern typography scaled for desktop and mobile reading contexts.

---

## 1. Typefaces

* **UI Interface Text:** **Inter** (Google Fonts) — Chosen for its high legibility, clean lines, and neutral rendering at varying weights.
* **Code Blocks:** **JetBrains Mono** — A monospace font designed for programmers, featuring distinctive character heights and wide symbols.

---

## 2. Type Scale Hierarchy

The scale below maps structural elements to corresponding CSS parameters:

| Font Name | Size | Weight | Line Height | Application Areas |
|---|---|---|---|---|
| **Display** | `36px` | `700` | `1.2` | Center welcome title (Welcome Screen). |
| **Heading 1** | `28px` | `700` | `1.25` | Main modal headings (AuthModal). |
| **Heading 2** | `22px` | `700` | `1.3` | Settings panel headings, secondary categories. |
| **Heading 3** | `18px` | `600` | `1.35` | Card grouping titles, sidebar headers. |
| **Body Large** | `16px` | `400` | `1.6` | Chat message outputs (User & Assistant). |
| **Body** | `15px` | `400` | `1.6` | Form descriptions, modal list parameters. |
| **Small** | `13px` | `400 / 500` | `1.5` | Sidebar navigation rows, button text. |
| **Micro** | `11px` | `400 / 600` | `1.4` | Input rules, timestamp indicators. |
| **Code** | `13px` (mono) | `400` | `1.6` | Inside programming code blocks. |
